12.3.1. リポジトリ¶

Groongaのリポジトリは GitHub 上にあります。次のコマンドでcloneできます。

% git clone --recursive https://github.com/groonga/groonga.git

Groonga関係のプロジェクト（grntest, fluent-plugin-groongaなど） のリストはリンク先をごらんください。
